# Vitality Protocol — Coaching Site

Marketing site for Vitality Protocol LLC, a high-ticket online coaching brand
built around "The Vitality Protocol" — a 4-pillar (Reveal, Restore, Rebuild,
Reinforce) methodology for fixing hormone issues and rebuilding fitness.

See [info.md](info.md) for full brand, audience, and messaging details.

## Structure

- `index.html`, `site/`, `deploy/`, `fresh/`, `vpsite/` — static site builds
- `app/` — React + TypeScript + Vite source for the site
- `assets/` — shared images and assets
